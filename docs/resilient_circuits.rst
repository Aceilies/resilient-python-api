resilient\_circuits package
===========================

This package is a framework for rapid development of Resilient Action Module event processors in Python.
It contains the :class:`resilient_circuits.actions_component.ResilientComponent` that handles events,
the :class:`resilient_circuits.actions_component.ActionMessage` that contains an Action Module event,
and other helpers.
