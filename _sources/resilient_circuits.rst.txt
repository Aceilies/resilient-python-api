resilient\_circuits package
===========================

This package is a framework for rapid development of Resilient Action Module functions and event processors in Python.
It contains the :class:`resilient_circuits.ResilientComponent` that custom components inherit,
the :class:`resilient_circuits.FunctionMessage` that represents a Function call event,
the :class:`resilient_circuits.ActionMessage` that represents an Action Module event,
and other helpers.
